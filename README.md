# 🐦 Flappy Bird Clone (Python Turtle Edition)

A fun desktop-based **Flappy Bird Clone** built using **Python** and the **Turtle graphics library**.  
This project recreates the classic gameplay with custom improvements like dynamic backgrounds, changing characters, and sound effects.

> 🎮 Developed as a creative attempt to build a game from scratch using Python.

---

## 📌 Project Overview

This project is inspired by the original Flappy Bird game but includes several custom modifications and enhancements.

Unlike the original version, this game:
- Changes the background dynamically
- Changes the player character design
- Includes sound effects
- Is built using Python’s Turtle graphics

Currently optimized for **laptop/desktop use**.

---

## 🚀 Features

- 🐤 Flappy Bird-style gameplay
- 🎨 Background changes dynamically
- 👾 Character changes during gameplay
- 🔊 Sound effects included
- ⌨️ Jump action using **Space key**
- 🖥️ Designed for laptop/desktop
- 🐍 Built using Python & Turtle (no game engine)

---

## 🛠️ Tech Stack

- Python 3
- Turtle Graphics Library
- Built-in Sound Modules (winsound)

## 🎯 How to Play

- Press **Spacebar** to make the bird jump.
- Avoid hitting the pipes.
- Try to achieve the highest score.
- Enjoy dynamic backgrounds and changing characters!
